# AdaptLink Feature Scope

The reference screenshot guides the shared UI style only. AdaptLink features come from our seller-intelligence logic.

## Pre-Launch Features

- Product idea selection and product details input
- Product category, target price, launch stock, and key features
- Market demand/readiness score
- Competition pressure and competitor snapshot
- Price and margin room
- Differentiation score
- Listing readiness score
- Launch recommendation
- Launch action plan
- OpenAI-supported reasoning for positioning, margin, and launch risk

## Post-Launch Features

- Seller existing product selection
- Timeframe selector for last 30 days, last 3 months, and last 6 months
- Sales, CTR, conversion rate, net margin, reviews, and competitor position
- Sales funnel with impressions, clicks, add-to-cart, orders, and conversion
- Sales trend chart
- Product health score
- Competitor benchmark
- Review theme analysis
- Review sentiment
- Buyer question insights
- Seller communication and response timing
- OpenAI action plan with next improvement areas

## API-Dependent Integrations

These are not removed from scope. They need credentials or approved access before official live use:

- Shopee Open Platform product, order, stock, and sales data
- Shopee ads/marketing data where permissions allow
- Seller-owned reviews and buyer-question/chat exports
- OpenAI API structured model calls and agent routing
- Direct Shopee write actions, only if explicitly approved later
