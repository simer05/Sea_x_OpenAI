# Seller Perspective Workstream

This folder contains the shared planning specs for Shopee Singapore seller-focused workflows using OpenAI.

## Parallel Ownership

- `01_seller_insights_spec.md`: Seller analytics, live data strategy, pricing, catalog, and competitor signals.
- `02_seller_ai_assistant_spec.md`: Seller assistant workflows, listing optimization, reply drafting, and automation support.

Both workstreams should use official or approved Shopee data sources where possible. OpenAI should be used for analysis, classification, drafting, summarization, and decision support after data is collected.
