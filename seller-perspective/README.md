# Seller Perspective

This folder is for the seller-side Shopee work. The split is simple: one person looks at seller insights, the other works on the assistant-style workflows.

## Ownership

- `01_seller_insights_spec.md`: Seller analytics, live data strategy, pricing, catalog, and competitor signals.
- `02_seller_ai_assistant_spec.md`: Seller assistant workflows, listing optimization, reply drafting, and automation support.

For data, start with mock files or approved Shopee API access. OpenAI comes after that: summaries, classification, drafts, and recommendations.
